# 🖥️ SYS IO

> ⚡ A powerful real-time system information tool for monitoring your PC hardware and performance.

## 📊 Overview

SYS IO is a lightweight system monitoring tool that provides real-time insights into hardware performance, system components, and running processes. It delivers a clear and structured overview of critical system data, including temperatures, network activity, and hardware identification.

## 🔍 Monitored Components

- 🧩 **Motherboard**
- 🧠 **CPU (Processor)**
- 🎮 **GPU (Graphics Card)**
- 💾 **RAM (Memory)**
- 📀 **Storage Devices**
- 🌐 **Network Activity**
- 🌡️ **Temperatures**
- ⚙️ **Running Processes**
- 🆔 **HWID (Hardware ID)**

## 🚀 Features

- ⚡ **Real-time monitoring** of system performance
- 📊 **Clean & structured UI** for better readability
- 🌡️ **Accurate temperature tracking**
- 🌐 **Live network insights**
- 🆔 **Hardware identification (HWID support)**
- 🪶 **Lightweight & fast performance**

## 🛠️ Programming Language's Used

Python, JavaScript, HTML, CSS
